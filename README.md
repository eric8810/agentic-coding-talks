# Agentic Coding 工程实践

关于 Agentic Coding 工程实践的演讲幻灯片，涵盖 Harness Engineering、Plan Mode、Design Mode、Code Mode 等核心理念与实践方法。

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Motion (Framer Motion)
- Lucide React (图标)
- @lobehub/icons (Coding Agent 品牌图标)

## 本地运行

**前置条件：** Node.js

1. 安装依赖：
   ```bash
   pnpm install
   ```
2. 启动开发服务器：
   ```bash
   pnpm dev
   ```
3. 浏览器访问 `http://localhost:3000`

## 操作方式

- **键盘：** 左右箭头 / 上下箭头 / 空格键翻页
- **触屏：** 左右滑动翻页
- **按钮：** 点击底部 Previous / Next 按钮

## 构建

```bash
pnpm build
```
